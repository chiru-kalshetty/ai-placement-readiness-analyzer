# AI-Assisted Placement Readiness Analysis and Recommendation System

## 📌 Overview
This project analyzes a student's placement readiness using academic and practical parameters.

## 🚀 Features
- Student performance scoring
- AI-based recommendations
- Dashboard analytics
- Leaderboard system
- Profile management

## 🛠️ Tech Stack
- Django
- HTML/CSS/JavaScript
- Chart.js
- SQLite

## 📊 How it Works
The system evaluates CGPA, projects, and internships to generate a readiness score and provides suggestions for improvement.

## 🔮 Future Scope
- Machine Learning integration
- Resume analysis
- Company-specific readiness scoring
