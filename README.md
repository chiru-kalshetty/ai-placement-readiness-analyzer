# AI-Assisted Placement Readiness Analysis and Recommendation System

## 📌 Overview
This project analyzes a student’s placement readiness using academic and practical parameters like CGPA, projects, and internships. It generates a readiness score and provides suggestions for improvement.

---

## 🚀 Features
- Student performance scoring  
- AI-assisted recommendations  
- Dashboard with analytics  
- Leaderboard system  
- Profile management  

---

## 🛠️ Tech Stack
- Django  
- HTML, CSS, JavaScript  
- Chart.js  
- SQLite  

---

## ⚙️ How it Works
The system evaluates:
- CGPA  
- Projects  
- Internships  

Based on these, it assigns a score and suggests improvements.

## ▶️ How to Run

1. Clone the repository  
2. Install dependencies
pip install -r requirements.txt
3. Run migrations  
python manage.py migrate
4. Start server  
python manage.py runserver
